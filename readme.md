# PHP Blog System

A simple yet powerful blog management system built with PHP and MySQL.

## ✨ Features

- 👤 User Authentication (Login/Register)
- 📝 Create, Edit, Delete blog posts
- 🖼️ Image upload with preview
- 📱 Responsive design
- 🔐 Secure password hashing
- 🎨 Modern UI with gradient design
- 📊 Admin dashboard
- 🖋️ Post author tracking

## 🛠️ Technologies Used

- PHP 7+
- MySQL
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## 📋 Requirements

- XAMPP/WAMP/LAMP server
- PHP 7.0 or higher
- MySQL 5.6 or higher
- Web browser

## 🚀 Installation

1. Clone the repository
```bash
git clone https://github.com/Mukeshmajhi001/Blog_post
```
Move to htdocs folder (XAMPP)
bash
cd C:/xampp/htdocs/
Import database

Open phpMyAdmin

Create database blog_db

Import database.sql file

Configure database

Open config.php

Update database credentials

Run the project

Open browser and go to http://localhost/php-blog-system
